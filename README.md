# E-Commerce Customer Cohort & Lifecycle Analytics Pipeline

An end-to-end data analytics architecture that extracts raw e-commerce transaction data from Microsoft SQL Server, builds optimized downstream business logic views, and surfaces interactive lifecycle insights inside an executive-ready Power BI dashboard.

## 🚀 Project Overview & Business Value
Acquiring new customers is highly expensive. This project delivers a scalable framework to analyze **customer retention decay** and **value segmentation**. By calculating monthly retention cohorts and classifying shoppers into behavioral buckets, marketing teams can identify precisely when churn happens and execute targeted re-engagement campaigns to maximize Customer Lifetime Value (CLV).

---

## 🛠️ Tech Stack & Architecture
* **Database Engine:** Microsoft SQL Server (SSMS)
* **Data Modeling & Analytics:** T-SQL (Common Table Expressions, Window Functions, Aggregate Views)
* **BI & Data Visualization:** Power BI Desktop (DAX Modeling, Matrix Heatmaps, Interactive Segment Distributions)
* **Version Control:** Git
